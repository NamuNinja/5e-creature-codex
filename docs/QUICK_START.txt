STEP 1: Get AdMob Set Up (15 minutes)
  1. Go to https://admob.google.com/
  2. Sign up for free account
  3. Create new app "5th Edition Compendium"
  4. Create 2 ad units: Banner + Interstitial
  5. Copy your Ad Unit IDs

STEP 2: Update Ad IDs (5 minutes)
  1. Open www/admob-config.js
  2. Replace TEST IDs with YOUR IDs
  3. Open AndroidManifest.xml
  4. Replace TEST App ID with YOUR App ID
  5. Save files

STEP 3: Build the App (1-2 hours first time)
  üìñ FULL INSTRUCTIONS: BUILD_INSTRUCTIONS.md

  Quick version:
  $ cd dnd-android-app
  $ npm install
  $ npx cap add android
  $ npx cap sync android
  $ npx cap open android

  Then build in Android Studio:
  Build > Generate Signed Bundle/APK > Android App Bundle

STEP 4: Submit to Play Store (1 hour)
  üìñ FULL INSTRUCTIONS: PLAY_STORE_SUBMISSION.md

  1. Create Play Console account ($25)
  2. Upload app bundle (AAB file)
  3. Add screenshots (take from emulator)
  4. Use provided descriptions
  5. Submit for review!
