### Step 1: Create Play Console Account
1. Go to https://play.google.com/console/
2. Pay $25 one-time registration fee
3. Complete account setup

### Step 2: Create New App
1. Click "Create app"
2. Enter app name: "5th Edition Creature Compendium"
3. Set default language: English (US)
4. Choose app/game: App
5. Choose free/paid: Free
6. Accept policies

### Step 3: Set Up Store Listing
1. Upload all graphics
2. Enter descriptions
3. Set category
4. Add tags
5. Provide contact details
6. Enter privacy policy URL

### Step 4: Content Rating
1. Complete questionnaire
2. Get rating certificate
3. Apply rating to app

### Step 5: App Content
1. Declare ads: YES
2. Target audience: 10+
3. Privacy policy: [YOUR URL]
4. Data safety: Complete form

### Step 6: Upload App Bundle
1. Production > Create new release
2. Upload AAB file
3. Set version name: 25.0.0
4. Set version code: 1
5. Add release notes

### Step 7: Pricing & Distribution
1. Choose countries (All or select)
2. Confirm free app
3. Acknowledge content policies

### Step 8: Submit for Review
1. Review all sections
2. Fix any errors
3. Submit for review
